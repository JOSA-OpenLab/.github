# JOSA OpenLab

**JOSA OpenLab** is a mentorship-driven open source initiative organized by the **Jordan Open Source Association (JOSA)**.  
The program enables developers, students, and researchers to gain real-world experience by contributing to impactful open source projects under professional guidance.

## About JOSA

The **Jordan Open Source Association (JOSA)** is a non-profit organization based in Jordan that promotes:

- Open Source Software and Hardware
- Open Standards and Open Data
- Digital Rights and Internet Freedom

JOSA works to strengthen the open source ecosystem through education, research, advocacy, and community engagement.

## About OpenLab

OpenLab is a structured apprenticeship and mentorship program focused on practical contribution rather than theoretical training.  
Participants collaborate in real development environments while following professional workflows and best practices.

Key aspects include:

- Real open source project contributions
- Mentor-led code reviews and feedback
- Collaborative development and documentation
- Community-driven learning

## Objectives

- Develop strong, real-world open source skills
- Encourage sustainable and ethical technology development
- Build contributors ready for global open source communities
- Support innovation through openness and collaboration
